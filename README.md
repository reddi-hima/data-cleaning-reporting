# 🧹 Data Cleaning & Reporting Automation

## 📌 Project Overview

The *Data Cleaning & Reporting Automation* project demonstrates how to clean, preprocess, and analyze raw datasets using *Power BI, **Python, and **Excel*. The project focuses on improving data quality by handling missing values, removing duplicate records, correcting inconsistencies, and generating interactive reports that support better business decisions.

This project showcases an end-to-end data preparation workflow and presents the cleaned data through an interactive Power BI dashboard with meaningful KPIs and visualizations.

---

## 🎯 Objectives

- Import raw employee data from CSV.
- Identify and remove duplicate records.
- Handle missing values.
- Standardize inconsistent data.
- Automate reporting using Power BI.
- Create interactive dashboards.
- Generate business insights through visual analytics.

- Missing values
- Duplicate records
- Inconsistent department names
- Blank salary fields
- Missing attendance values
- Formatting inconsistencies

These issues are intentionally included to demonstrate data cleaning techniques.

## 🔄 Project Workflow

### Step 1 – Import Data
- Load CSV dataset into Power BI.
- Verify column names and data types.

### Step 2 – Data Cleaning
- Remove duplicate rows.
- Handle missing values.
- Standardize department names.
- Correct data formats.
- Validate employee information.

### Step 3 – Data Transformation
- Change data types.
- Create calculated columns.
- Build DAX measures.
- Prepare data for reporting.

### Step 4 – Dashboard Development
Develop an interactive dashboard using Power BI.
